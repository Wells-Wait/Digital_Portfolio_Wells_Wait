Current Research

Emotion-Inspired Spiking Neural Network using Temporary Dynamic Routing of Pathways and Synaptic Strengths During Inference with Heterogeneous Outputs

Yeah, that title is long and hard to read. It is a work in progress. The basic idea is that I am removing the layer structure on a spiking neural network. This removal of structure allows for less direct and longer paths through limited nodes(neurons), allowing for more computation with fewer nodes. While this is fascinating, the real goal behind this modification was to allow for pathways to change during inference. Changing values is easy, but the number of connections is more difficult, leading to the use of a spiking network, and to add variety to the connections, there is no predetermined path to an output. As such, changing connections is possible, just leaving how to implement the changes. To accomplish this goal, nodes outside standard operation are added to act as 'emotion' triggers, which temporarily change the pathways. The training, of course, may be challenging. More will be added about this project as things move forward.
